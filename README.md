![](screenshot.png)

## How to
- Run portwarden-gui.exe
  - Create backup
    - Input an encryption password
    - Click Backup
    - Login with your bitwarden account
  - Decrypt backup
    - Input the encryption password
    - Select the backup to decrypt
    - Click Decrypt
- Wait for the process to finish
